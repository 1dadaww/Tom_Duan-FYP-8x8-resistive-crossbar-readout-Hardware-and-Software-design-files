# Tom Duan FYP: 8x8 Resistive Crossbar Readout

This public repository contains the hardware and software design files for an 8x8 resistive crossbar readout system.

## Repository Files

- `esp32_firmware.zip`  
  ESP32 firmware for hardware control and data acquisition.

- `fundamental_readout_matlab.zip`  
  MATLAB scripts for measurement, calibration, reconstruction, and analysis.

- `PCB_Project_fyp - voltage_mode - Copy (2026-6-13 16-38-03).zip`  
  Altium source files for the main crossbar readout PCB.

- `FYP_Breakoutboard - Copy (2026-6-13 16-40-43).zip`  
  Altium source files for the 8x8 resistor-array board.

- `FYP_Breadout2_Calibration - Copy (2026-6-13 16-40-02).zip`  
  Altium source files for the diagonal calibration board.
